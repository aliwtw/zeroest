Binary Fundamentals

What is Binary?

Definition

The binary numeral system, also referred to as base-2, is a positional number system that utilizes exactly two symbols: {0, 1}. Each digit, called a bit, occupies a position corresponding to a power of two, analogous to the base-10 system in which digits represent powers of ten.

A number base (or radix) determines the number of unique digits, including zero, used to represent numbers. For example, base-2 (binary) uses only 0 and 1, while base-10 (decimal) uses digits from 0 to 9.
Numbers with there bases can be can be represented as: $\text{Number}_\text{Base}$
Example: Binary: $0101_2$, Decimal: $5_{10}$

Formally, an $n$-bit binary number can be expressed as: $$ b_{n-1}b_{n-2}\dots b_1b_0 $$ where each bit $b_i \in \{0, 1\}$ for all $i \in \{0, 1, \dots, n-1\}$. The corresponding integer value (in base-10) is given by the weighted sum: $$ \sum_{i=0}^{n-1} b_i \cdot 2^i $$ The bit $b_0$ is referred to as the least significant bit (LSB), as it corresponds to the lowest positional weight, $2^0$. In contrast, the bit $b_{n-1}$ is the most significant bit (MSB), associated with the highest positional weight, $2^{n-1}$.

Example

Convert the binary number $1011_2$ to its decimal (base-10) representation:

$$ \begin{aligned} 1011_2 &= (1 \times 2^3) + (0 \times 2^2) + (1 \times 2^1) + (1 \times 2^0) \\ &= 8 + 0 + 2 + 1 \\ &= 11_{10} \end{aligned} $$

Why Binary?

Compatibility with Boolean Algebra

Binary logic aligns directly with Boolean algebra - the foundation of digital logic. Operations like AND, OR, NOT, and XOR are naturally defined over binary inputs, enabling the construction of efficient and scalable logic circuits.

Efficient Hardware Implementation

Binary logic gates are simple to implement using transistors. This results in compact, power-efficient circuits that scale effectively, supporting advancements in microprocessor design and integrated systems.

Formal Integration with Finite Fields

Binary operations correspond to arithmetic in the finite field $\mathbb{F}_2$, which is fundamental in cryptographic constructions, error-correcting codes, and formal verification methods. This algebraic structure provides a rigorous framework for secure digital communication.

Deterministic Computation Models

Binary supports deterministic models such as finite automata and Turing machines. These models form the theoretical foundation of computer science, demonstrating that binary is sufficient for expressing any computable function within a formal system.

Physical Alignment with Digital Electronics

Binary representation is inherently suited to physical hardware, where two distinct voltage levels represent logical 0 and 1 (commonly 0V and a positive voltage like 3.3V or 5V). This direct mapping simplifies circuit design and minimizes signal ambiguity.

Noise Tolerance and Signal Stability

With only two discrete states, binary systems are more resistant to electrical noise and voltage fluctuations than multi-level systems. This improves reliability and reduces the chance of signal misinterpretation in digital circuits.

Basic Binary Units

Bit

A bit (b) (short for binary digit) is the fundamental unit of binary information. It holds one of two values: 0 or 1. In algebraic terms, a bit belongs to the finite field $\mathbb{F}_2$, which contains exactly two elements.

Byte

A byte (B) consists of 8 bits and is a standard unit for data storage and memory access: 8-dimensional vector over the binary field

$$ \text{Byte} = (b_7, b_6, b_5, b_4, b_3, b_2, b_1, b_0) \in \mathbb{F}_2^8 $$
  • Represents one ASCII character.
  • CPU architectures commonly operate on byte-aligned data.
  • Maximum value (unsigned): $2^8 - 1 = 255$

Nibble

A nibble is a group of 4 bits-exactly half a byte: $\text{Nibble} = (b_3, b_2, b_1, b_0) \in \mathbb{F}_2^4$

  • Capable of representing a single hexadecimal digit (0–15).
    Example: $ 0000_2 = 0_{16}, \quad 1111_2 = F_{16}$
  • Used in compact digital representations such as 7-segment displays.

Word

A word is a fixed-length sequence of bits processed as a unit by a CPU. The word size defines the computational width of a system:

  • Common sizes: 32-bit and 64-bit.
  • Defines processor architecture and affects performance, memory addressing, and instruction sets.

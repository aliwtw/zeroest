Binary Positional Value and Negative Powers of Two

In a binary number, each bit has a weight determined by its position relative to the binary point. The value of each digit is calculated as:

$$ \text{Value} = d_i \cdot 2^i $$

where $d_i \in \{0, 1\}$ is the bit at position $i$. If the bit is to the **left** of the binary point, $i$ is non-negative (e.g., $2^3$, $2^0$); if it is to the **right**, $i$ is negative (e.g., $2^{-1}$, $2^{-3}$).

Positional Impact in a Binary Number

Example: Binary number 1011.101

Bit Position Power of 2 Value
1 3 $2^3$ 8
0 2 $2^2$ 0
1 1 $2^1$ 2
1 0 $2^0$ 1
. Binary Point
1 −1 $2^{-1}$ 0.5
0 −2 $2^{-2}$ 0
1 −3 $2^{-3}$ 0.125

Summing the values:

$$ 8 + 0 + 2 + 1 + 0.5 + 0 + 0.125 = 11.625 $$

So, 1011.101 (binary) = 11.625 (decimal).

Why Negative Powers Matter

Negative powers of two represent fractional values. Each position after the binary point halves the contribution of the bit: 

Position:    -1     -2     -3     -4     -5
Power:     2⁻¹   2⁻²   2⁻³   2⁻⁴   2⁻⁵
Decimal:   0.5   0.25  0.125 0.0625 0.03125

This system allows binary to represent fractional numbers with high precision—although not all decimal fractions can be represented exactly in binary (e.g., $0.1$ becomes repeating).

Example: Decimal 0.625 to Binary (Base-2)

To convert the fractional part of a decimal number to binary, we repeatedly multiply the fraction by 2. The integer part of the result becomes the next binary digit. This process continues until the fractional part becomes 0 (or until desired precision is reached).

The method can be summarized as:

$$ \text{Binary Digit} = \text{Integer Part of } ( \text{Fraction} \times 2 ) $$
Step Fraction × 2 Result Integer Part (Bit) New Fraction
1 0.625 0.625 × 2 1.25 1 0.25
2 0.25 0.25 × 2 0.5 0 0.5
3 0.5 0.5 × 2 1.0 1 0.0

Reading the integer parts in order gives the binary representation of the fractional part: $$0.625_{10} = 0.101_2$$

Thus, if the full decimal number were \( 5.625_{10} \), we would convert the integer part \(5\) to binary

Summary

- Binary digits left of the point: powers of 2 (2⁰, 2¹, 2², …)
- Binary digits right of the point: negative powers of 2 (2⁻¹, 2⁻², …)
- Each position affects the total value exponentially
- Understanding position is crucial in float calculations
